### 项目简介

一个类似于音乐软件歌词展示的组件

## 主要功能

1、歌词单行渐变；

2、歌词多行渐变；

3、拖动进度，同步更新歌词进度；

## 未来要加入的功能

1、即将播放句子"..."提示；

2、放大正在播放句子字体；

3、自定义字体大小；
